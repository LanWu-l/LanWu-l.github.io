# Lan Wu Personal Website

Static academic website modeled on a compact GitHub Pages personal-site layout.

## Included files

- `index.html` — home/about page
- `research.html` — publications, working papers, and ongoing projects
- `teaching.html` — teaching and conference activity
- `awards.html` — awards and service
- `style.css` — site styling
- `Lan_Wu_CV.pdf` — linked CV

## Deploy on GitHub Pages

1. Create a repository named `<your-github-username>.github.io`.
2. Upload all files in this folder to the repository root.
3. In GitHub, go to **Settings → Pages**.
4. Set the source to the main branch and root folder.
5. Visit `https://<your-github-username>.github.io/` after GitHub Pages finishes building.

## Notes to edit before publishing

- Replace the initials avatar with a photo if desired.
- The phone number from the CV is visible on the home page. Remove it from `index.html` if you do not want it public.
- Add Google Scholar, LinkedIn, or GitHub links to the navigation/profile panel if needed.
